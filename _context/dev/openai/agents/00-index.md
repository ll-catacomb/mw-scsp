Agents Docs Index
=================

Brief summaries of each guide in this folder.

- [01-guides-agents.md](01-guides-agents.md) — High-level overview of Agents and AgentKit: build with Agent Builder, deploy with ChatKit, and optimize with evals/trace grading.
- [02-guides-agent-builder.md](02-guides-agent-builder.md) — Visual Agent Builder guide: compose workflows with nodes, preview/debug, publish, and deploy via ChatKit or SDK code.
- [03-guides-node-reference](03-guides-node-reference) — Reference for all Agent Builder nodes: core (Start, Agent, Note), tools (File search, Guardrails, MCP), logic (If/else, While, Human approval), and data (Transform, Set state).
- [04-guides-agent-buider-safety.md](04-guides-agent-buider-safety.md) — Safety guidance for agent workflows: prompt injection, data leakage, and mitigations (structured outputs, approvals, guardrails, evals, model choices).
- [05-guides-agents-sdk.md](05-guides-agents-sdk.md) — Safety in building agents (duplicate of 04); discusses risks and mitigations when designing workflows.
- [06-guides-chatkit.md](06-guides-chatkit.md) — ChatKit overview and quickstart: embed the chat UI, create sessions, integrate React/Web Component, and iterate with themes, widgets, and actions.
- [07-guides-chatkit-themes.md](07-guides-chatkit-themes.md) — Theming and customization: control color scheme, typography, density, start screen, header buttons, attachments, and @mentions/entities.
- [08-guides-chatkit-widgets.md](08-guides-chatkit-widgets.md) — Widgets guide and reference: design with the Widget Builder, upload assets, handle actions, and use the component/container API.
- [09-chatkit-actions.md](09-chatkit-actions.md) — Actions in ChatKit: trigger from widgets or client, handle on server or client, strongly typed actions, and form payload patterns.
- [10-guides-custom-chatkit.md](10-guides-custom-chatkit.md) — Advanced self-hosted ChatKit: implement the server, endpoints, data/file stores, client tools, widgets, and actions.
- [11-guides-agent-evals.md](11-guides-agent-evals.md) — Agent evals overview: when to use Datasets vs Evals, prompt optimizer, and cookbook resources; recommends trace grading.
- [12-guides-trace-grading.md](12-guides-trace-grading.md) — Trace grading: annotate and score traces, then run evaluations across runs to benchmark and improve agents.
- [13-guides-voice-agents.md](13-guides-voice-agents.md) — Voice agents: speech-to-speech vs chained architectures, Realtime API transports, design and prompting guidance, and example demos.
