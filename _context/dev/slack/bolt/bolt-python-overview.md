# Bolt for Python Documentation Overview

## Introduction

"Bolt for Python is a Python framework to build Slack apps with the latest Slack platform features." The framework enables developers to create applications leveraging current Slack capabilities, with a quickstart guide available for initial setup.

## Core Documentation Structure

The documentation includes several key sections:

**Building & Development**
- Building an App
- Slack API calls and message sending
- Events and message listening
- App UI & Interactivity
- AI integration in applications
- Custom Steps implementation

**Configuration & Architecture**
- App Configuration and Socket Mode
- Middleware & Context management
- Lazy listeners for Function-as-a-Service (FaaS)
- Adaptors for various platforms
- Authorization and OAuth security

**Additional Resources**
- Tutorial content including AI chatbot examples
- API Reference documentation
- Legacy documentation for older patterns

## Getting Help & Support

The framework provides multiple support channels:

- **Issue Tracker**: GitHub repository for questions, bug reports, and feature requests
- **Email Support**: Developer support available at support@slack.com
- **Documentation**: Comprehensive guides plus an in-depth reference section

## Contributing

The documentation and codebase are open source, welcoming community contributions through the published Contributor's Guide on GitHub.
