# Welcome

This is your documentation home page. This template includes a dynamic documentation system that reads markdown files from the `_content/` folder.

## Features

- **Automatic routing** &mdash; Any folder in `_content/` becomes a route
- **MDX support** &mdash; Use React components in your markdown
- **Navigation** &mdash; Built from `_meta.json` files
- **Print view** &mdash; All docs on a single page at `/docs/print-all`

## How it Works

1. Create markdown files in `_content/docs/` (or any subfolder name)
2. Add a `_meta.json` to control navigation order
3. Access your docs at `/docs` (or `/<folder-name>`)

## Adding More Documentation

To add a new documentation section:

1. Create a folder: `_content/guides/`
2. Add markdown files and a `_meta.json`
3. Access at `/guides`
